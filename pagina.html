Programação Front-end

HTML

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para estruturar páginas na web.

CSS

CSS (Cascading Style Sheets) é responsável pela estilização das páginas HTML.

Bootstrap

Bootstrap é um framework front-end que facilita a criação de interfaces modernas e responsivas.

JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web.

PHP

PHP é uma linguagem de programação utilizada no desenvolvimento back-end.

Informações Extras
